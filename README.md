# 🚗 Hyundai Sales Dashboard (Power BI)

## 📊 Project Overview
This project is a Power BI dashboard built to analyze Hyundai car sales data. It provides insights into sales performance, customer behavior, and regional trends.

## 📌 Key Features
- Total Sales, Units Sold, Average Selling Price, Transactions (KPI Cards)
- Sales Channel Analysis (Dealer, Online, Corporate)
- Fuel Type Analysis (Petrol, Diesel, EV)
- Monthly Sales Trend
- Top 5 States by Sales
- Interactive Filters (Car Model, Month, State, Variant, Payment Mode)

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
